MONGODB ‚Äì INTRODUCTION

MongoDB (atlas)
MongoDB is a document database designed with applications in mind. their cloud version is called MongoDB Atlas

Cluster
a cluster is what runs the database and houses the data.

to connect to an mongodb cluster in python you need the Mongodb_uri and provide it to the the MongoClient function to set up the client

Example
import os
from pymongo import MongoClient

MONGODB_URI = os.environ["MONGODB_URI"]

# Initialize a MongoDB Python client
mongodb_client = MongoClient(MONGODB_URI, appname="datacamp_text_to_query_course")

# Check the connection to your cluster
ping_result = mongodb_client.admin.command("ping")
print(ping_result)

Aggregation pipelines
You can also perform complex data manipulations, such as grouping, sorting, transforming, and joining data across collections using the MongoDB Query API.
This is done using aggregation pipelines. Aggregation pipelines in MongoDB consist of a series of stages, each transforming the data in some way and passing the results on to the next stage.
these often than take own library key value pairs as their values

when using operators that group in any way like group sum unwind the field name needs to have an addition $ before it

Operator

Use
$match
 filters documents to pass only that match the given query
$group
 gropu by
$project
 reshapes selected field allowing to add remove and rename fields
$sort
 order by
$limit
 passes a specific maximum number of documents
$unwind
 decostructs an array field from input documents to ouput one document for each element of the array
$loopup
 left outer join
$addfields/ $set
 adds new fields or replaces the value of exsitign field
$skip
 skips a specified number of documetns
$count
 returns singe document copntaining the count of the input documents for that stage
$out
 writes the results of the aggegration pipeline to a new collection or replaces the cotnent of an existing one (must be last age)
$merge
 writes and merges the result into an existing collection (must be the last stage)

Example
query = [ { "$unwind": "$directors" },
{ "$sort": { "released": -1 }}, { "$limit":5 }, {"$project": {"title": 1, "_id": 0 }}]

Collection
A collection is the same thing as table in other databases.Collections consist of flexible JSON-like documents that can support varying key-value pairs or even nested structures, instead of rows with fixed columns like you'd have in a relational database.

to access a collection after connection to a db cluster you can use just db["collection name"]

Example
db = mongodb_client["sample_mflix"]
collection = db["movies"]

Filtering a collection
to filter mongoDB you need to write a query in teh form of a dictionary. containing key value pairs on which you want to filter.

Example single filters
query = {"genre": "Romance"}
using the $and operator and providing it with a list of single filters you can create and and filter.

List all collections
The list_collection_names can be used to generate a list of all collections (tables) that are available in a database

Example
import os
from pymongo import MongoClient

MONGODB_URI = os.environ["MONGODB_URI"]
# Initialize a MongoDB Python client
mongodb_client = MongoClient(MONGODB_URI)

# Access the sample_mflix database
db = mongodb_client["sample_mflix"]

db.list_collection_names()
