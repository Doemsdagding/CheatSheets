Shell
shell is accessed using a command-line terminal interface like alacritty. ach time a command is entered, the shell runs some other programs, prints their output in human-readable form, and then displays a prompt to signal that it's ready to accept the next command

Directory navigation and manipulation

pwd
pwd is used to retrieve the current directory.

Example
pwd #/home/repl

ls
ls is used to list what is inside a directory, if not given a directory it list the current directory its contents
in some shell you need to use ls . to get current directory.

when not starting with / it will use a relative path when starting with a / it will use an abosulate path.

adding a -R flag will result in ls also displaying the underlaying direcotries their files

adding -F flag will add an / after directory and a * after an executable

Example
ls #backup bin course.txt  seasonal
ls seasonal
#autumn.csv spring.csv summer.csv winter.csv
ls /home/repl/seasonal
#autumn.csv spring.csv summer.csv winter.csv
ls -R -F   # seasonal/ course.text
./seasonal:
autumn.csv spring.csv asummer.csv

cd
to move between directories cd is used
by doing cd .. you move one directory above
using cd ~ brings you back to the home directory

Example
cd /home/repl
cd seasonal
cd ..
cd ~

cp
cp is used to copy paste a file by cp <source path><file.extention> <target path> 
by also adding a filename to the target you can rename the file durring the copy
cp <source path><file.extention> <target path><file.extention>

you can also use multiple source files and copy them to the same target directory
cp <file.extention> <file.extention> <target path>

Example
cp original.txt duplicate.txt
cp seasonal/autumn.csv seasonal/winter.csv backup

Directory navigation and manipulation 2

mv
mv is used to move a file and works just like cp.
mv <file.extention> <target path>

it can also be used to rename a file within a director
further mroe it can be used to move and or rename entire directories

Example
# movign a file
mv seasonal/spring.csv  backup
#renaming a file
mv course.txt old-course.txt

rm
rm is used to delete a file and like with mv and cp you can provide multiple files at the same time

Example
rm autumn.txt
rm seasonal/summer.csv

rmdir
rmdir is used to delete directories

Example
rmdir people

mkdir
mkdir is than used to create a directory mkdir <directory name>

Example
mkdir yearly

man
to know what a command does you can use man <command> and it will return the manual for the function

Example
man head
#HEAD(1)               BSD General Commands Manual              HEAD(1)

NAME
     head -- display first lines of a file

SYNOPSIS
     head [-n count | -c bytes] [file ...]

DESCRIPTION
     This filter displays the first count lines or bytes of each of
     the specified files, or of the standard input if no files are
     specified.  If count is omitted it defaults to 10.

     If more than a single file is specified, each file is preceded by
     a header consisting of the string ``==> XXX <=='' where ``XXX''
     is the name of the file.

SEE ALSO
     tail(1)

Files in shell

cat
cat is used to print the contents of files inside the terminal.

Example
cat file.txt # introduction to file ....

less
by default cat uses more but when starting with less the cocntents of a file gets printed one page at a time.

spacebar =you can continue the file
q = quit and return to the terminal
:n = move to the next file when you gave it multiple files

Example
less seasonal/spring.csv seasonal/summer.csv | cat

head
head is used to display by default the first 10 lines of a file.
when adding the -n <number> flag you can have head print less or additional lines

Example
head seasonal/summer.csv
head -n 5 seasonal/winter.csv

tail
tail is the opposie of head and will print the last rows of a document

Example
tail seasonal/summer.csv

cut
cut is used to retrieve columns from a column based file
it is structured using flags with -f indicating the fields with - from and , additional fields and -d used to indicatd the deliminter.
downside is that it breaks when delimiters are used in strings even when their quoted
Example
cut -f 2-5,8 -d , values.csv

history
using history will return a list of recent commands made each with an id.
using !<id> will execute the same command iwth that id again
